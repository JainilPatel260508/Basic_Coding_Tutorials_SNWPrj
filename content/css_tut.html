Welcome to CSS Basics

Welcome to the CSS Basics Course! CSS stands for Cascading Style Sheets, and it's what makes websites look beautiful and visually appealing. If HTML is the skeleton of a website, then CSS is the clothing and makeup that makes it attractive.

In this course, you'll learn how to transform plain HTML pages into stunning, professional-looking websites. We'll start with the basics like colors and fonts, then move on to more advanced topics like layouts and positioning. By the end of this course, you'll be able to style any website beautifully!

1. Introduction to CSS

CSS is the language that describes how HTML elements should be displayed on the screen. There are three ways to add CSS to your HTML pages, but we'll focus on the most common and recommended method.

Three Ways to Add CSS

1. Inline CSS - Adding styles directly to HTML elements:

<p style="color: blue; font-size: 20px;">This is blue text.</p>

2. Internal CSS - Using a style tag in the head section:

<head>
    <style>
        p {
            color: blue;
            font-size: 20px;
        }
    </style>
</head>

3. External CSS - Creating a separate CSS file (recommended):

<head>
    <link rel="stylesheet" href="style.css">
</head>

Basic CSS Syntax

CSS rules consist of a selector and a declaration block:

p {
    color: blue;
    font-size: 16px;
}
  • Selector - Points to the HTML element you want to style (here it's "p" for paragraphs)
  • Property - The aspect you want to change (like color or font-size)
  • Value - The setting for that property (like blue or 16px)
  • Declaration - Each property-value pair, separated by semicolons
Best Practice: Use external CSS files for larger projects. It keeps your HTML clean and makes it easier to maintain consistent styling across multiple pages.

2. CSS Selectors

Selectors are how you target specific HTML elements to apply styles. Think of them as a way to point at exactly what you want to style on your page.

Element Selector

Targets all elements of a specific type:

p {
    color: green;
}

h1 {
    font-size: 32px;
}

This will style ALL paragraphs and ALL h1 headings on your page.

Class Selector

Targets elements with a specific class attribute. Classes start with a dot (.):

.highlight {
    background-color: yellow;
}

.important {
    font-weight: bold;
    color: red;
}

Use it in HTML like this: <p class="highlight">This is highlighted</p>

ID Selector

Targets a single element with a specific ID. IDs start with a hash (#):

#header {
    background-color: navy;
    color: white;
}

#footer {
    text-align: center;
}

Use it in HTML like this: <div id="header">Header Content</div>

Important: Each ID should be unique on a page (use it only once), but you can use the same class on multiple elements. Classes are more flexible and reusable.

3. Colors and Backgrounds

Colors bring life to your website. CSS offers several ways to specify colors, and you can apply them to text, backgrounds, borders, and more.

Ways to Specify Colors

1. Color Names - Simple and easy to remember:

p {
    color: red;
    background-color: lightblue;
}

2. Hexadecimal - Six-digit codes starting with #:

h1 {
    color: #FF0000;  /* Red */
    background-color: #0000FF;  /* Blue */
}

3. RGB - Red, Green, Blue values (0-255):

div {
    color: rgb(255, 0, 0);  /* Red */
    background-color: rgb(0, 0, 255);  /* Blue */
}

Text Color

p {
    color: blue;
}

h1 {
    color: #333333;
}

Background Color

body {
    background-color: #f0f0f0;
}

.box {
    background-color: white;
}
Pro Tip: Make sure there's enough contrast between text color and background color so your content is easy to read. Dark text on light backgrounds or light text on dark backgrounds works best.

4. Text and Font Styling

Typography is crucial for readability and visual appeal. CSS gives you complete control over how text appears on your website.

Font Family

Changes the typeface of your text:

body {
    font-family: Arial, sans-serif;
}

h1 {
    font-family: 'Times New Roman', serif;
}

Always provide fallback fonts in case the first choice isn't available on the user's device.

Font Size

Controls how big or small your text appears:

p {
    font-size: 16px;
}

h1 {
    font-size: 32px;
}

small {
    font-size: 12px;
}

Font Weight

Makes text bold or light:

.bold {
    font-weight: bold;
}

.light {
    font-weight: 300;
}

.normal {
    font-weight: normal;
}

Text Alignment

Aligns text left, right, center, or justified:

.center {
    text-align: center;
}

.right {
    text-align: right;
}

.justify {
    text-align: justify;
}

Text Decoration

Adds underlines, overlines, or removes default decorations:

a {
    text-decoration: none;  /* Removes underline from links */
}

.underline {
    text-decoration: underline;
}
Readability Tip: Use a font-size of at least 16px for body text to ensure easy reading on all devices. Line spacing (line-height) of 1.5 to 1.8 also improves readability.

5. The Box Model

Every HTML element is essentially a rectangular box. Understanding the CSS box model is crucial for controlling layout and spacing on your web pages.

Understanding the Box Model

The box model consists of four parts, from inside to outside:

  • Content - The actual content (text, images, etc.)
  • Padding - Space between content and border (transparent)
  • Border - A line around the padding and content
  • Margin - Space outside the border (transparent)
.box {
    width: 300px;           /* Width of content area */
    padding: 20px;          /* Space inside the box */
    border: 2px solid black; /* Border around the box */
    margin: 10px;           /* Space outside the box */
}

Width and Height

Control the size of the content area:

.container {
    width: 800px;
    height: 400px;
}
Important: By default, when you set width and height, you're only setting the size of the content area. Padding and borders add to the total size. Use box-sizing: border-box; to include padding and border in the width/height calculation.

6. Borders and Spacing

Borders and spacing properties help you create visual separation and organize your content effectively.

Borders

Add borders around elements:

.box {
    border: 2px solid black;
}

.thick-border {
    border: 5px solid blue;
}

.dashed {
    border: 1px dashed red;
}

You can also set borders for individual sides:

.custom {
    border-top: 3px solid blue;
    border-bottom: 1px solid gray;
    border-left: none;
    border-right: none;
}

Padding

Adds space inside an element, between content and border:

.padded {
    padding: 20px;  /* All sides */
}

.custom-padding {
    padding-top: 10px;
    padding-right: 20px;
    padding-bottom: 10px;
    padding-left: 20px;
}

/* Shorthand */
.shorthand {
    padding: 10px 20px;  /* top/bottom, left/right */
}

Margin

Adds space outside an element, creating distance from other elements:

.spaced {
    margin: 20px;  /* All sides */
}

.centered {
    margin: 0 auto;  /* Centers block elements */
}

.custom-margin {
    margin-top: 30px;
    margin-bottom: 10px;
}
Remember: Padding adds space inside (affects background), margin adds space outside (always transparent). Use padding when you want the background color to extend, use margin to create space between elements.

7. Working with Layouts

Creating layouts is about arranging elements on your page. CSS provides several properties to control how elements are positioned and how they interact with each other.

Display Property

Controls how an element is displayed:

.block {
    display: block;  /* Takes full width, starts on new line */
}

.inline {
    display: inline;  /* Only takes necessary width */
}

.inline-block {
    display: inline-block;  /* Combines both behaviors */
}

Float Property

Makes elements float to the left or right:

.left {
    float: left;
    width: 50%;
}

.right {
    float: right;
    width: 50%;
}

Text Alignment

Aligns content within an element:

.center-text {
    text-align: center;
}

.right-text {
    text-align: right;
}

Creating a Simple Two-Column Layout

.container {
    width: 90%;
    margin: 0 auto;
}

.column {
    width: 45%;
    float: left;
    margin: 10px;
    padding: 20px;
    background-color: white;
}
Layout Tip: When using floats, you might need to clear them to prevent layout issues. Add a container with clear: both; or use overflow: hidden; on the parent element.

8. Practice Your Skills

Now it's time to put everything you've learned into practice! The best way to master CSS is by styling real web pages.

Challenge: Style Your HTML Page

Take an HTML page you created earlier and apply CSS styling. Try to include:

  • A consistent color scheme (choose 2-3 main colors)
  • Different font sizes for headings and paragraphs
  • Background colors for different sections
  • Proper padding and margins for spacing
  • Borders around certain elements (like boxes or images)
  • Centered text for headings
  • Styled links (remove underline, change colors on hover)
  • A simple two-column layout using float

Example Complete CSS Style

Here's a simple, complete example to get you started:

body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    margin: 0;
    padding: 0;
}

.header {
    background-color: #333;
    color: white;
    padding: 20px;
    text-align: center;
}

.container {
    width: 90%;
    margin: 20px auto;
    background-color: white;
    padding: 20px;
}

h1 {
    color: #333;
    font-size: 32px;
}

p {
    line-height: 1.6;
    color: #666;
}

.button {
    background-color: blue;
    color: white;
    padding: 10px 20px;
    border: none;
}

Tips for Success

  • Start with basic colors and fonts, then add more complex styling
  • Test your CSS in a browser frequently to see the changes
  • Use browser developer tools (F12) to experiment with styles
  • Keep your CSS organized with comments
  • Don't be afraid to try different values and see what happens
  • Look at websites you like and try to recreate their styling
Next Steps: Once you're comfortable with CSS basics, you can explore advanced topics like responsive design (making websites work on mobile devices), CSS animations, and modern layout techniques like Flexbox and Grid. Keep practicing and experimenting!
